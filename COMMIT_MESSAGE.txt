Implements comprehensive options trading system for tip tokens as specified in issue #200.

Features:
- Define option contracts (Call and Put options)
- Implement option pricing using simplified Black-Scholes model
- Add exercise functionality with atomic settlement
- Handle option expiration with collateral return
- Track option positions per address

Core Components:
- options/mod.rs: Core data structures, storage management, and position tracking
- options/pricing.rs: On-chain pricing engine with intrinsic and time value calculation
- options/exercise.rs: Exercise, settlement, expiration, and cancellation logic

Contract Functions:
- Admin: init_options_trading, update_option_pricing
- Trading: write_option, buy_option, exercise_option, expire_option, cancel_option
- Query: get_option, get_written_options, get_held_options, get_option_position, get_active_options
- Utility: calculate_option_premium, batch_expire_options

Security:

- Atomic settlement to prevent partial execution
- Access control (only holders can exercise, only writers can cancel)

- Pause support
